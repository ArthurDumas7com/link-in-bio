# Таам Артур — link in bio

Однофайловая link-in-bio страница в стиле bento-grid.

**Стек:** vanilla HTML + CSS + JS, без сборки и зависимостей (только шрифт `Bricolage Grotesque` с Google Fonts).

## Запуск

Открыть `index.html` в браузере. Никакого сервера не требуется.

## Деплой через GitHub Pages

После пуша в репозиторий:

1. Открыть **Settings → Pages** в репозитории на GitHub.
2. В разделе **Build and deployment** выбрать:
   - **Source:** `Deploy from a branch`
   - **Branch:** `main` (или `master`) / `/ (root)`
3. Через 1–2 минуты страница будет доступна по адресу:
   `https://<username>.github.io/<repo-name>/`
